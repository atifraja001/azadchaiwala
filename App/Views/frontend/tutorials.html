HTML Intro

HTML is the markup language use to create web pages

What is HTML?

  • HTML stands for Hypertext Markup Language
  • The markup language use to create web pages
  • It tells the browser how to load and display the page
  • HTML consists of series of tags

How simple HTML document will look like


Example


<!DOCTYPE html> <html> <head><title>Title</title></head> <body> <h1>My First Heading</h1> <p>My First Paragraph</p> </body> </html>

Try it Yourself

Example Explained

  • The <!DOCTYPE html> defines that it is the HTML 5 document.
  • The <html> element is the starting tag of an HTML document.
  • The <head> has the meta information of the web page.
  • The <title> shows the title of the web page<./li>
  • The <body> is the main body of html document which contains all the tags
    such as paragraphs, headings, images, videos, articles.
  • The <h1> defines the largest headings (there are 6 tipes of headings).
  • The <p> is a paragraph element.
